Copy and paste these commands one by one into your terminal:

1. Stop the dev server if running (press Ctrl+C)

2. Install the missing packages:

npm install @radix-ui/react-avatar@1.1.2 @radix-ui/react-label@2.1.1 @radix-ui/react-tabs@1.1.2 @radix-ui/react-tooltip@1.1.6 @radix-ui/react-dialog@1.1.4 @radix-ui/react-dropdown-menu@2.1.4 @radix-ui/react-progress@1.1.1 @radix-ui/react-select@2.1.4 @radix-ui/react-slider@1.2.1 @radix-ui/react-switch@1.1.2

3. Start the dev server:

npm run dev

-----------

If that doesn't work, try this:

rm -rf node_modules package-lock.json
npm cache clean --force
npm install
npm run dev
